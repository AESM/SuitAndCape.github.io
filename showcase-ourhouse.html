Housemates doing house things.

Source Link

OurHouse is a mobile and web social media app for housemates, allowing users to coordinate social events, household chores, and payments among their fellow housemates.
The functionality of the app is contained within a sleek and intuitive interface, divided into four main sections.

Home: Respond to and delete tasks, notifications, and activity invitations from other housemates, with support for filtering by message type.

Message: Send any type of message to the other members of the house.

Payments: Connect to Venmo and send money to any of the house members.

Housemates: Add new housemates and view the profiles and message history of existing housemates.

Info

Front-End:

Back-End:
Ruby on Rails, ActiveRecord, Rails API

Additional:
Decoupled Architechture, Auth0, Google+ API, Venmo API

Team:
Brian Vogelgesang, Joseph Won, Ali Esmaili, Charles Kim, & Anthony Ng

Notes

My teammates and I have all experienced living with housemates. We wanted to solve the issue of prioritizing communication: activities, scheduling, expenses, and discussing household needs. This is what inspired us to create OurHouse.

OurHouse was built using a decoupled architecture: the front-end with Angular, Ionic, JavaScript, jQuery, HTML5, and CSS3; and the back-end using Rails, ActiveRecord, and Rails API with features from Auth0, the Google+ API, and the Venmo API.
