OurHouse:

This iOS and Android mobile app makes coordinating social events, household chores, and payments among your fellow housemates incredibly simple. Using a decoupled architecture, the front-end was built with Angular, Ionic, JavaScript, jQuery, HTML5, and CSS3; while the back-end was built using Rails, Auth0, the Google+ API, and the Venmo API.

Through the Home tab, the user can respond to or delete messages, and filter messages by type: tasks, notifications, and activitiy invitations. Unread messages are marked.

Through the Message tab, a user can send any of type of message to the other members of the House. Through the payments tab, a user can connect to Venmo and send money to any of the House members.

The Housemates tab allows the user to view every housemate's profile. A message histroy dashboard can be opened by clicking the profile picture, and adding new housemates via email is easy.
